# Sistema de Gestión de Taller Mecánico MV

Aplicación web para gestión de taller mecánico con Next.js y PostgreSQL.

## Deploy en Vercel

El proyecto está configurado para deployarse en Vercel con PostgreSQL (Neon).

### Variables de entorno requeridas:
- `DATABASE_URL`: Conexión a PostgreSQL

### Configuración:
- Root Directory: `taller-app`
- Framework: Next.js
- Node Version: 20
